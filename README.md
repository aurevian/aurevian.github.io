# Personal Academic Website

This repository contains the source code for my personal academic website and research portfolio. The website is designed to present my background, research interests, technical skills, and selected projects in a clear and accessible format.

The main purpose of this site is to serve as a professional digital profile where visitors can quickly understand my research direction, technical stack, and ongoing work. In particular, it is intended to highlight my focus on quantum computing, quantum machine learning, scientific computing, and computational research.

The website also functions as a central hub for connecting my broader research ecosystem. Alongside my profile and technical background, it is designed to provide access to external resources such as GitHub repositories, project pages, and Jupyter Book materials where my more detailed theoretical and computational work is documented.

## Main Features

- Personal introduction and academic profile
- Research-oriented portfolio structure
- Links to GitHub, LinkedIn, contact information, and external research resources
- Sections for programming languages, tools, and technical competencies
- Extendable structure for projects, publications, and Jupyter Books

## Technologies

This website is built using:

- Jekyll
- HTML / CSS / JavaScript
- GitHub Pages and Jupyter Book
